# Sketchpad (Etch-a-Sketch)

This project is part of **The Odin Project – Foundations** curriculum.  
The goal was to build an interactive drawing grid using **JavaScript**, practicing DOM manipulation, event handling, and dynamic styling.

**Live Demo:**  https://codebymihaly.github.io/Sketchpad/

## Features

- Interactive drawing grid  
- Adjustable grid size  
- Hover‑based drawing   
- Simple and clean UI  
- Dynamic DOM manipulation  


## What I used

- **HTML5** – structure  
- **CSS3** – layout and styling  
- **JavaScript (ES6)** – DOM manipulation, events  


##  What I Learned

- Creating elements dynamically with JavaScript  
- Handling mouse events  
- Updating styles through JS  
- Managing state inside a simple app  
- Building interactive UI without frameworks
